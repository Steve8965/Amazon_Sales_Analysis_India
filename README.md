# Amazone_Sales_Analysis_India
Sales analysis project using Excel (Power Query) and Power BI on Amazone India dataset (Mar - Jun 2022)
# Amazon Sales Analysis (India)

## 📊 Project Overview
This project analyzes Amazon sales data in India from March to June 2022.

## 🛠 Tools Used
- Excel (Power Query)
- Power BI

## 📈 Key Insights
- Strong sales performance with peak in April
- Sales concentrated in specific product categories
- Majority of sales from non-B2B customers
- Regional disparities in sales distribution

## 📂 Files
- Power BI dashboard (.pbix)
- Cleaned dataset (.xlsx)
- Project report (.pdf)

## 🎯 Objective
To clean, analyze, and visualize sales data for business decision-making.
